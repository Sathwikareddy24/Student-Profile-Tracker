
Student Profile & Hobby Tracker:
A simple Python CLI application built to manage student information and track extracurricular interests. This project focuses heavily on List Mutability and Data Type Handling.

Features:
Dynamic Profile Creation: Captures student names and academic grades with proper type casting.

Hobby Management: Uses advanced list methods to add, insert, and remove items dynamically.

Data Slicing: Automatically extracts "Top Picks" from a collection of interests.

Clean Formatting: Utilizes string multiplication and the .format() method for a structured console output.

Concepts Used: 
This project is a practical application of the first 23 modules of my Python curriculum:

Integer Conversion: Converting string inputs to numbers for data integrity.

List Manipulation: Using .append(), .insert(), and .remove() to manage collections.

Indexing & Mutability: Modifying specific list items by their position.

Slicing: Using the [start:end] syntax to retrieve sub-sections of data.

Length Function: Measuring list size using len().
